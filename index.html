Research: Modeling Human Overthinking Through Decision Delay Data

Sunkenapally Subhash ‚Äî M.Sc. (AI & DS)

A behavioral data-science research project that introduces a computational framework to measure human overthinking using Decision Delay Data (DDD). The study proposes two new metrics:

  • Overthinking Ratio (OR): Measures cognitive inefficiency by relating decision accuracy to decision time.
  • Overthinking Threshold (OT): Identifies the point where additional deliberation reduces decision accuracy.

Key results and methods: temporal feature engineering, correlation analysis (r = -0.63), polynomial regression showing an inverted‚ÄëU relationship, and K‚Äëmeans clustering that classifies users into impulsive, balanced, and overthinkers. Average OT ‚âà 1.45 √ó EDT.

Example: computing Overthinking Ratio (Python)

def compute_or(accuracy, adt, eps=1e-6):
    # OR = (1 - accuracy_normalized) / (adt + eps)
    # accuracy assumed 0..1
    return (1 - accuracy) / (adt + eps)

# usage
or_val = compute_or(0.75, 42)
print('OR:', or_val)
